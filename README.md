# WoWOffsetDumper

Automatically dumps offsets and descriptors for World of Warcraft BfA.

## Getting Started

Open in Visual Studio 2017 and build.

### Prerequisites

* [Visual Studio 2017](https://visualstudio.microsoft.com/downloads/)
* World of Warcraft (duh)
